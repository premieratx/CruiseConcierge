## System Architecture (Technical Implementation)

### Frontend
- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui components
- Wouter for routing

### Backend
- Express + Node.js
- PostgreSQL database
- Stripe payment integration
- Mailgun email delivery
- GoHighLevel SMS notifications

### Key Technical Features
- **Xola Integration**: Embedded booking widgets for 6 products (3 boats x 3 disco packages) via iframe approach
- **Quote Builder**: Lovable app integration at `/chat` route
- **Admin Dashboard**: Lead management, calendar booking, quote generation
- **Real-time Availability**: Google Sheets integration for tracking
- **Database Keepalive**: Pings every 4 minutes to prevent dormancy
- **Crash Prevention**: 4-layer protection (process handlers, Express middleware, UptimeRobot monitoring, build validation)
- **Schema Markup**: Comprehensive SEO with 28 schema files for AI/search optimization
- **Authentication**: Passport.js with PostgreSQL session store

### UX Components
- **TableOfContents**: Sticky sidebar navigation
- **StickyCTA**: Sticky mobile CTA bottom bar
- **LazyImage**: Intersection Observer lazy loading
- **CollapsibleSection**: Responsive accordion (desktop open, mobile collapsible)
- **VideoTestimonials**: YouTube video grid with modal playback
- **TransportationGuide**: Hotel and ride-share information

### Pricing Validation
- **CRITICAL**: Private cruise package add-ons (Essentials and Ultimate) are FLAT FEES per cruise, not hourly rates
- Server-side validation prevents pricing tampering
- Transparent breakdowns for taxes, gratuity, and deposits

### External Dependencies
- **Stripe**: Payment processing
- **GoHighLevel**: SMS & lead management
- **Google Sheets**: Availability tracking
- **Mailgun**: Email delivery
- **OpenRouter**: AI chatbot interactions
- **Replit DB**: WordPress blog storage
- **Xola**: Online booking widgets
